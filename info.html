Info about the music player

This is a semi-advanced music player built in JavaScript and HTML. The premise is simple: upload a file, and listen.

This is NOT a YouTube/SoundCloud/Spotify player, but it supports any file your browser does.

Why?

I wanted to combine all of my previous music-related projects into one: Visualizer, Player, and my in-the-works full-stack local file aggregator and player. In other words, it's < audio > on steroids.

How?

Any song you upload is entirely local and never leaves your device. The Web Audio API is utilized to display visualizers and an equalizer. Lyrics are fetched from LRClib. Metadata is extracted using jsmediatags.
